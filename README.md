# コマンドとか

## 開発環境

nodejs 10.15.1
ui-spec-md https://github.com/pxgrid/ui-spec-md

### セットアップ

```
npm i
```

### ui-spec 生成

```
npm run generate:ui-spec
```

### ui-spec 編集サーバー立ち上げ

```
npm run edit
```
